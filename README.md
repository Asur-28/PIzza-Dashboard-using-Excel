# 🍕 Pizza Sales Dashboard

An interactive Microsoft Excel dashboard built to analyze pizza sales data. The dashboard provides insights into revenue, sales trends, customer behavior, product performance, and ordering patterns using Pivot Tables, Pivot Charts, Slicers, and Timeline Filters.

---

## 📌 Project Overview

This project analyzes pizza sales data and presents key business metrics through an interactive Excel dashboard. It helps users identify sales trends, peak order times, best-selling pizzas, and category-wise performance, making data-driven decision-making easier.

---

## 🎯 Objectives

- Analyze overall pizza sales performance.
- Monitor revenue and order trends.
- Identify best and worst-selling pizzas.
- Compare sales by category and size.
- Understand customer ordering patterns.
- Build an interactive dashboard for business insights.

## 📈 Dashboard Features

- Revenue Analysis
- Daily Order Trend
- Hourly Order Trend

- Sales by Pizza Size
- Top 5 Best Selling Pizzas
- Bottom 5 Worst Selling Pizzas
- Interactive Slicers
- Timeline Filter
- Dynamic KPI Cards

---

## 🔍 Key Insights

- Friday recorded the highest number of orders.
- Peak ordering hours are between **12 PM–1 PM** and **5 PM–8 PM**.
- Classic Pizza category generated the highest sales.
- Large-size pizzas contributed the most revenue.
- Pepperoni Pizza is the best-selling pizza.
- Brie Carre Pizza is the lowest-selling pizza.
